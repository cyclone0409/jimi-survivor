把你的音频素材放在这个目录，并使用下面这些文件名替换占位文件：

- `home-bgm.wav` / `home-bgm.m4a` / `home-bgm.mp3`：开始页面背景音乐。
- `bgm-1.m4a` / `bgm-1.wav` / `bgm-1.mp3`：游戏内普通 BGM 第 1 首。
- `bgm-2.m4a` / `bgm-2.mp3` / `bgm-2..wav` / `bgm-2.wav`：游戏内普通 BGM 第 2 首，手机端建议使用 m4a 或 mp3。
- `bgm-3.m4a` / `bgm-3.mp3` / `bgm-3.wav`：游戏内普通 BGM 第 3 首，手机端建议使用 m4a 或 mp3。
- `bgm-4.m4a` / `bgm-4.wav` / `bgm-4.mp3`：游戏内普通 BGM 第 4 首。
- `hao-bgm.m4a` / `hao-bgm.mp3` / `hao-bgm.wav`：自在极意豪形态专属 BGM，手机端建议使用 m4a 或 mp3。
- `gameover-bgm.m4a` / `gameover-bgm.wav` / `gameover-bgm.mp3`：失败页面 BGM，进入失败页时播放，点击“三气归来！”后停止。
- `attack.wav`：玩家自动攻击时播放。
- `hit.wav`：玩家子弹命中敌人时播放。

普通 BGM 目前按 `bgm-1` 到 `bgm-4` 顺序轮播。替换音频时，优先直接覆盖现有同名文件。
